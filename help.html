Help for using the RPG Web Client
The buttons on the right
Numbered Scene Buttons - clicking these will change your current scene. The view will change to the scene you have selected, and catch you up.
Sheet - shows you your character sheet and current status
Game Info - shows you some nice background info on the game
Map - links you to the maps for the game
Pinterest - links you to the Pinterest group for the game
Help - This page, so you know about this one
Logs - Opens the log viewer
Chr - This button allows you to change to another character on the server (or another form for the same chr.)
How to make rolls
Key to playing in an RPG is making rolls.
To make rolls, you type a command in the main window like this:
roll PER v 5
This is rolling your score in PER (Personalilty) vs a difficulty of 5
Both sides of the roll can be a number, a score or a combination, such as
PER+5 v 10+3
